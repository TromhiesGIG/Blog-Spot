The Basic Language of the Web: HTML

Posted by Laura Jones on Monday 21st 2027

All modern websites and web applications are built using three fundamental technologies: HTML, CSS and Javascript. These are the languages of the web. In this post, let's focus on HTML. We will learn what HTML is all about, and why you too should learn it.

What is HTML

HTML stands for HyperTect Markup Language. It's a markup language thst web developers use to structure and describe the content of a webpage (not a programming language).

HTML consists of elements that describe different types of content: paragraphs, links, heading, images, video, etx. Web browsers understand HTML and render HTML code as websites.

In HTML, each element is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual content

You can learn more atMDN Web Docs

Why you should learn HTML

There are countless reasons for learning the fundamental language of the web. Here are 5 of them:

  • To be able to use the fundamental web dev language
  • To hand-craft beautiful website instead of relying on tools like WordPress or Wix
  • To build web applications
  • To impress friends
  • To have fun
